# Common Plots Library

This folder contains minimal, clean implementations of common plots using matplotlib and seaborn.

## Available Plots

1. **barplot.py** - Bar chart
2. **heatmap.py** - Heatmap visualization
3. **histogram.py** - Histogram
4. **scatter_plot.py** - Scatter plot
5. **line_plot.py** - Line plot with legend
6. **boxplot.py** - Box plot
7. **violin_plot.py** - Violin plot
8. **pie_chart.py** - Pie chart
9. **area_plot.py** - Area plot
10. **distribution_plot.py** - Distribution plot with KDE
11. **pairplot.py** - Pair plot
12. **count_plot.py** - Count plot
13. **regression_plot.py** - Regression plot

All plots include:

- Title
- X and Y labels
- Legends (where applicable)
- Clean, minimal code
- Grid (where appropriate)

## Usage

Simply run any plot file:

```bash
python plots/barplot.py
```
